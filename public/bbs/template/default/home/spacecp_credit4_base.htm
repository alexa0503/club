1. 风迷币定义

会员通过指定的消费或参加指定会员活动获得的可用于兑换消费的奖励。

2. 风迷币来源

推荐购车、购买原厂配件、维修保养等指定消费及参加公司指定的会员活动。

3. 风迷币使用

网上商城兑换礼品、精品配件及电子抵用卷（可用于在经销商处消费）。风迷币不得转让、兑现。

4. 风迷币有效期

在获取当年及下一个自然年内有效
